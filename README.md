# QRCodeGenerator

一个简单的二维码生成工具，使用 [ZXing.Net](https://github.com/micjahn/ZXing.Net) 实现，支持自定义图标，输入文本即可生成二维码，右键单击生成的二维码即可弹出菜单【复制】，【修改图标】，【保存】。
